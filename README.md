# React + Vite + GoIT

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

# npm module

1. npm install modern-normalize
2. npm install clsx
3. npm install react-icons

# start in laptop

1. "vite": "^4.4.0"
2. npm cache clean --force
3. npm install
4. npm run dev
5. npm install formik
